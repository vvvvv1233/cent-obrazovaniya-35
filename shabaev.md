Проект: Сайт "Центр образования №35"
Описание проекта
Одностраничный сайт (landing page) для МАОУ "Центр образования №35" (г. Уфа).
Сайт содержит информацию о школе, её достижениях, мероприятиях, контактах.

Структура сайта
Шапка — название школы
Приветственный блок — миссия, преимущества, кнопки навигации
Достижения — результаты 2024–2025 учебного года
Мероприятия — ближайшие события с датами
Подвал — адрес, телефон, email, политика конфиденциальности

Технологии
HTML5
CSS3 (адаптивная вёрстка, Flexbox/Grid)
Git / GitHub
